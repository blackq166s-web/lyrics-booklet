# Layout and Style Guide

Use this guide when choosing the visual direction for a booklet.

## Presets

| Preset | Best for | Visual tone |
| --- | --- | --- |
| `gallery` | polished album booklets, gifts, portfolio PDFs | warm paper, editorial, quiet accent color |
| `minimal` | laser printing, rehearsal binders, dense lyric packs | clean black and white, high legibility |
| `noir` | dramatic album packaging and dark covers | black cover, gold accent, white body pages |
| `zine` | indie demos, fan-made booklets, rough-cut drafts | small-press energy, warmer paper, stronger accent |

## Language Count

- Use `columns` for 1 to 3 languages.
- Use `stacked` for 4 or more languages, or when lines are long.
- Use `--languages en,zh,ja` to reorder or hide languages for a specific export.

## Typography

- Keep the cover/title font expressive, and keep lyric fonts boring enough to read.
- Prefer Noto families for broad script coverage.
- For CJK languages, use Google Fonts family names when using the downloader: `Noto Sans SC`, `Noto Sans TC`, `Noto Sans JP`, `Noto Sans KR`.
- Use `Inter` for English and Spanish.
- For Arabic, Hebrew, Thai, Hindi, and other additional scripts, set each language font in `LANGUAGES`.

## Print Notes

- A4 output is the default because it works well for home and office printers.
- Leave a wider left margin when the pages will be stapled or spiral bound.
- Use `minimal` when ink usage matters.
